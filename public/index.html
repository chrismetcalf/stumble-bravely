Stumble Bravely Baltimore!

Drunk in Baltimore? Don't know whether the neighborhood you're in is safe? Happen to know the zip code of that neighborhood? Text Stumble Bravely to find out whether you should take a cab or not!

(410) 205-4320

This service is run as a demo application for Baltimore, MD's data.baltimorecity.gov open data portal, and the Socrata Open Data API. The code is also available via Github. This app is not affiliated officially with the City of Baltimore government, and should not be considered authoritative. If you get mugged or walk off a pier, its your fault, not ours. Drink safely!
